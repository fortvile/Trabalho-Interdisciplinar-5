# [SINTRA]

O projeto SINTRA – Sistema Inteligente de Tráfego tem como objetivo desenvolver e validar um modelo de controle adaptativo de semáforos baseado em comunicação em rede. A proposta busca integrar conceitos de Redes de Computadores e Sistemas Operacionais, aplicados a um ambiente de simulação e, futuramente, em prototipação real.

O sistema utiliza clientes para enviar informações de tráfego a um servidor central, responsável por processar os dados e ajustar dinamicamente o tempo de abertura dos semáforos. Dessa forma, é possível reduzir o tempo médio de espera dos veículos e otimizar o fluxo viário.
